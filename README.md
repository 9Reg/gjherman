# GJ Herman Static Author Site

This is a plain static website for `gjherman.com`. It uses regular HTML and CSS, plus one dependency-free Python script that builds the blog pages from dated folders.  And has no cookies!

## Local Blog Workflow

Create a new folder under `content/blog` using a date-first slug:

```text
content/blog/2026-07-10-post-title/
  post.md
  hero.jpeg
```

The `post.md` file needs front matter like this:

```markdown
---
title: Post Title
date: 2026-07-10
summary: One or two sentences for the blog grid.
image: hero.jpeg
image_alt: Description of the article image
---

Article body goes here.
```

The blog markdown also supports three safe content classes. Put the class marker on its own line directly before the paragraph or block:

```markdown
{.center}
Centered text

{.small}
A quieter note or image credit.

{.pullquote}
A larger emphasized quotation or key line.
```

Then rebuild the static blog:

```bash
python3 scripts/build_blog.py
```

The script asks before deleting and rebuilding the generated `blog/` directory. For a non-interactive rebuild, run:

```bash
python3 scripts/build_blog.py --yes
```

The generated files appear in `blog/`. Upload the whole project to static hosting, including `assets`, `blog`, `books`, `content`, `impressum`, and `index.html`.

## Pages

- `index.html` is the main author page.
- `books/index.html` is the longer book/manuscript page.
- `blog/index.html` and `blog/*/index.html` are generated by Python.
- `impressum/index.html` contains legal placeholders and image attributions.

## Book Page Cover Classes

Each book section can use a faded background image regardless of whether it has cover art. Add `with-cover` to sections that include a cover image aside, and `without-cover` to sections that should use only the text treatment over the background image.
